# Carbon Emissions Forecasting Project

## Overview

This project analyzes historical carbon emissions data and builds forecasting models to predict future emissions. The objective is to explore trends in emissions and evaluate time-series models for forecasting future carbon output.

Two forecasting approaches are implemented and compared:

* **SARIMA (Seasonal AutoRegressive Integrated Moving Average)**
* **Prophet (by Meta)**

The models are used to forecast emissions for **Scope 1 and Scope 2** categories and compare their predictive performance.

---

## Project Objectives

* Analyze historical carbon emissions data
* Perform data cleaning and exploratory analysis
* Build time-series forecasting models
* Compare forecasting performance across models
* Export model predictions for further analysis

---

## Dataset

The dataset contains historical emissions values categorized into:

* **Scope 1 emissions** – Direct emissions from owned or controlled sources
* **Scope 2 emissions** – Indirect emissions from purchased electricity, heat, or steam

Typical structure of the dataset:

| Year | Scope1 | Scope2 |
| ---- | ------ | ------ |
| 2015 | ...    | ...    |
| 2016 | ...    | ...    |
| 2017 | ...    | ...    |

---

## Models Implemented

### 1. SARIMA

SARIMA is a classical statistical time-series model that accounts for:

* Autoregression
* Differencing
* Moving averages
* Seasonal patterns

Separate SARIMA models were trained for:

* Scope 1 emissions
* Scope 2 emissions

---

### 2. Prophet

Prophet is a forecasting tool developed by Meta that models time-series data using:

* Trend components
* Seasonality
* Holiday effects
* Robust handling of missing data and outliers

Prophet models were also trained separately for:

* Scope 1 emissions
* Scope 2 emissions

---

## Forecast Output

The predictions generated by both models are exported to:


Example structure:

| Date | SARIMA_Scope1 | SARIMA_Scope2 | Prophet_Scope1 | Prophet_Scope2 |
| ---- | ------------- | ------------- | -------------- | -------------- |
| 2023 | ...           | ...           | ...            | ...            |
| 2024 | ...           | ...           | ...            | ...            |
| 2025 | ...           | ...           | ...            | ...            |

This file allows easy comparison of predictions across models.

## Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/carbon-emissions-forecasting.git
```

Navigate to the project directory:

```bash
cd carbon-emissions-forecasting
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

## Running the Project

Open the Jupyter notebook:

```
notebooks/carbon_emissions_analysis.ipynb
```

Run the notebook to:

1. Load and preprocess the emissions dataset
2. Train SARIMA and Prophet models
3. Generate forecasts
4. Export predictions to `forecast_output.csv`

## Evaluation

Model performance can be evaluated using metrics such as:

* RMSE (Root Mean Squared Error)
* MAE (Mean Absolute Error)

These metrics help determine which model provides more accurate forecasts.
